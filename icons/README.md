# icons フォルダについて

`アイコン.png`（オリジナル画像）から生成した PWA 用アイコンです。

| ファイル名 | サイズ | 用途 |
| --- | --- | --- |
| icon-192.png | 192x192 | 通常アイコン（purpose: any） |
| icon-192-maskable.png | 192x192 | マスク対応アイコン（purpose: maskable、周囲に余白あり） |
| icon-512.png | 512x512 | 通常アイコン（purpose: any） |
| icon-512-maskable.png | 512x512 | マスク対応アイコン（purpose: maskable、周囲に余白あり） |

アイコンを変更する場合は、元画像（正方形・1000px以上推奨）を用意して同じファイル名・サイズで上書きしてください。maskable版は周囲に約10%の余白を持たせて生成しています。
